# Black Box — Incubate Nepal '26

Website for Team Black Box, an interpretable AI research group under Incubate Nepal 2026, investigating shortcut learning and trustworthy deployment in medical imaging.

## Structure

- `index.html` — homepage: mission, team, news, research agenda
- `people.html` — full team directory
- `projects.html` — the 6 research threads
- `thread-*.html` — individual thread write-ups
- `research.html` — research approach and methods
- `reading.html` — reading group papers
- `style.css` — shared styles
- `assets/` — team photos

Static site, no build step. Open `index.html` directly or serve the folder with any static file server.
